⚡ Key Issue #133 Insight: Self-Hosted Runners at Scale

At 5,000 pipeline runs/day, GitHub-hosted runners cost $14,400/mo ($172,800/yr). Self-hosted runners attached to the existing Cloud Run OrchestratingAgent fleet cost $0 incremental. Trigger point: switch at >500 runs/day. Below that threshold, GitHub-hosted runners on a public repo cost $0.
