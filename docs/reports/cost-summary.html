⚡ Key Issue #134 Insight: DRE Tests Are Free to Run in CI

The DRE integration tests use stubbed model endpoints, making the CI runtime cost $0/mo. The 530 min/month of test execution fits comfortably within GCP Cloud Build's 3,600 min/mo free tier. The only production cost is the N=3 model API calls (~$34/mo at standard scale, reducible to ~$12/mo with prompt caching).

Running DeterministicLayer gates in-process (Python function calls within the OrchestratingAgent Cloud Run container) vs. spawning separate CI/CD jobs saves $77,844/year at edge scale (5,000 pipeline runs/day). At standard scale (50 runs/day), in-process gates cost $0 incremental — absorbed into the $1.73/mo Cloud Run container budget.
